## Enunciado
Escreva um programa em C# que solicite ao usuário que adivinhe um número secreto entre 1 e 100. O programa deve gerar aleatoriamente o número secreto e fornecer dicas sobre se o palpite do usuário é maior ou menor do que o número secreto. O programa deve continuar pedindo palpites até que o usuário adivinhe corretamente o número secreto.

## Extra
- Foi adicionado um contador de quantos chutes o usúario fez até acertar
